## Quick Start

1. **Development Mode:**
   ```bash
   cd /tmp/appex-portfolio
   npm install
   npm run dev
   ```

2. **Production Build:**
   ```bash
   npm run build
   ```

3. **Serve Built Version:**
   ```bash
   python3 serve.py
   ```
   Then visit: http://localhost:8000
